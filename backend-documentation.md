- **Create report**

    POST: /report

    BODY PARAMS:
    ```
    {
    "report_type":"Lab report",
    "description":"Lab report ordered by Dr.Raymond Brown",
    "idPatient": 31
     }
    ```

    RESPONSE:
    ```
     {
        "msg": {
            "description": "Lab report ordered by Dr.Raymond Brown",
            "idPatient": 31,
            "idReport": 12,
            "report_type": "Lab report"
        },
        "status": true
    }
    ```
